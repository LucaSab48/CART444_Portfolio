Practice

Work shaped by montage, interface logic, and speculative image-making.

My practice moves between filmmaking, game development, computational art, and design research. I am interested in how technical systems can produce atmosphere, structure attention, and reshape the way images are encountered.

Across short films, prototypes, and media experiments, I build projects that balance clarity with ambiguity, using code and editing as parallel tools for composition.
